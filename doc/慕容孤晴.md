# 慕容孤晴 Mod 文件夹格式化映射

## 源结构

```
慕容孤晴/
├── 3.6.1/
│   ├── BepInEx.zip                    ← BepInEx 框架，跳过
│   ├── BepInEx二创植物（3.6.1）/       ← 全部 mod zip
│   │   ├── (金银钻)杨桃.zip
│   │   ├── 比基尼海滩三兄弟.zip
│   │   └── ... (共 26 个)
│   └── 使用教程.txt                    ← 跳过
├── 3.7/
│   ├── BepInEx.zip
│   ├── BepInEx二创植物（3.7）/
│   │   ├── (金银钻)杨桃.zip
│   │   └── ... (共 24 个，比 3.6.1 少 2 个)
│   └── 使用教程.txt
```

## 版本号

| 目录名 | 规范化版本 |
|---|---|
| `3.6.1` | `3.6.1` |
| `3.7` | `3.7` |

版本号直接从目录名获取。

## 映射规则

### Zip → 植物MOD

源路径模式：`{版本}/BepInEx二创植物（{版本}）/*.zip`

| 源路径 | 目标路径 |
|---|---|
| `3.6.1/BepInEx二创植物（3.6.1）/(金银钻)杨桃.zip` | `慕容孤晴-3.6.1/植物MOD/(金银钻)杨桃/` |
| `3.6.1/BepInEx二创植物（3.6.1）/比基尼海滩三兄弟.zip` | `慕容孤晴-3.6.1/植物MOD/比基尼海滩三兄弟/` |
| ... (共 26 个) | ... |
| `3.7/BepInEx二创植物（3.7）/(金银钻)杨桃.zip` | `慕容孤晴-3.7/植物MOD/(金银钻)杨桃/` |
| ... (共 24 个) | ... |

**规则**：每个 zip 解压到 `{zip名不含扩展名}/` 目录下，该目录名即为 MOD 名称。

### 跳过的内容

| 源路径 | 原因 |
|---|---|
| `{版本}/BepInEx.zip` | BepInEx 框架，通用前置 |
| `{版本}/使用教程.txt` | 非 mod 文件 |
