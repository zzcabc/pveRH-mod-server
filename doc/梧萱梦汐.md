# 梧萱梦汐 Mod 文件夹格式化映射

## 源结构

梧萱梦汐是最大的作者文件夹，结构已半接近目标格式：

```
梧萱梦汐/
├── 3.6.1（新lib20260602，...长描述...）/     ← 版本 3.6.1
│   ├── .【置顶】MOD前置文件/                   ← BepInEx 框架，跳过
│   ├── MOD植物/          → 植物MOD
│   │   ├── 奥林匹斯级航空母舰/   (含 dll 等)
│   │   ├── 超级机械高坚果/
│   │   │   ├── 白鱼版/          (变体版本)
│   │   │   └── 林秋版/
│   │   └── ... (~199 个)
│   ├── MOD僵尸/          → 僵尸MOD (~16 个)
│   ├── MOD皮肤/          → 皮肤MOD (空目录)
│   ├── MOD关卡/          → 关卡 (~4 个)
│   ├── MOD插件/          → 其他 (~5 个)
│   ├── MOD铲子/          → 修改器 (~1 个)
│   ├── MOD手套/          → 修改器 (~1 个)
│   ├── MOD迷你宠物/      → 植物MOD (~2 个)
│   └── 修改器/           → 修改器 (~2 个 + 使用说明书跳过)
├── 3.7/                                    ← 版本 3.7
│   ├── .【置顶】MOD前置文件（注意！需要装里面的前置文件！）/
│   ├── MOD植物/          → 植物MOD (~197 个，少量增减)
│   └── ... (同上结构)
```

## 版本号规范化

| 原始目录名 | 规范化版本 |
|---|---|
| `3.6.1（新lib20260602，如发现问题...）` | `3.6.1` |
| `3.7` | `3.7` |

规则：取开头的数字版本号 `3.6.1` / `3.7`。

## 分类映射

源分类目录名 → 目标 MOD 类型：

| 源目录 | 目标类型 | 说明 |
|---|---|---|
| `MOD植物` | `植物MOD` | 含变体子目录 |
| `MOD僵尸` | `僵尸MOD` | 含变体子目录 |
| `MOD皮肤` | `皮肤MOD` | 当前为空 |
| `MOD关卡` | `关卡` | |
| `MOD插件` | `其他` | 插件/工具 |
| `MOD铲子` | `其他` | 铲子工具 |
| `MOD手套` | `其他` | 手套工具 |
| `MOD迷你宠物` | `植物MOD` | 无独立宠物分类 |
| `修改器` | `修改器` | 排除"使用说明书" |
| `.【置顶】MOD前置文件` | **跳过** | BepInEx 框架 |

## 操作规则

**操作类型**：文件已解压，执行**目录移动**（非 zip 解压）。

### 普通 MOD（无变体子目录）

将源 mod 目录直接重命名到目标路径：

```
源: {版本}/MOD植物/奥林匹斯级航空母舰/
目标: 梧萱梦汐-3.6.1/植物MOD/奥林匹斯级航空母舰/
```

### 变体 MOD（含子目录）

mod 名 = `{父目录}_{子目录}`：

```
源: {版本}/MOD植物/超级机械高坚果/白鱼版/
目标: 梧萱梦汐-3.6.1/植物MOD/超级机械高坚果_白鱼版/

源: {版本}/MOD僵尸/黄金丑跳/炼狱版/
目标: 梧萱梦汐-3.6.1/僵尸MOD/黄金丑跳_炼狱版/
```

**变体识别**：mod 目录下包含子目录（而非直接包含 dll 文件）。

### 跳过的内容

| 路径 | 原因 |
|---|---|
| `.【置顶】MOD前置文件/` | BepInEx 框架，用户自行安装 |
| `修改器/使用说明书/` | 非 mod 文件 |
