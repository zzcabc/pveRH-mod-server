# 高数羽衫 Mod 文件夹格式化映射

## 版本号规范化规则

| 原始版本名 | 规范化版本       | 说明 |
|---|-------------|---|
| `3.6（兼容3.6.1）` | `3.6.1`     | 3.6 版本，兼容 3.6.1 |
| `3.7` | `3.7`       | |
| `3.6.1` | `3.6.1`     | |
| `2.3` / `2.4` / `2.5` / `2.7` / `2.8` | 不保留3.6之前的版本 | 融合Mod更多锤子的版本 |
| `3.1.1` | 不保留3.6之前的版本     | |
| 无版本号 | `通用`        | 皮肤、周年庆单品等未标注版本的 mod |

## 目标格式

```
高数羽衫/高数羽衫-{版本}/{MOD类型}/{MOD名称}/{MOD文件}
```

MOD 类型固定为：`植物MOD` `僵尸MOD` `皮肤MOD` `修改器` `关卡`

## 一、1.修改器 → 修改器

源路径模式：`1.修改器/{版本}/{平台}/*`

| 源路径 | 目标路径 | 操作  |
|---|---|-----|
| `1.修改器/3.6.1/PC/PvZRHModfiedFor3.6.1【2026-5-24更新】.zip` | `高数羽衫-3.6.1/修改器/PvZRHModfiedFor3.6.1/` | 不解压 |
| `1.修改器/3.6.1/其他/其他3.6.1.APK` | `高数羽衫-3.6.1/修改器/其他3.6.1.APK` | 删除  |
| `1.修改器/3.7/PC/PvZRHModfiedFor3.7【2026-6-7更新】.zip` | `高数羽衫-3.7/修改器/PvZRHModfiedFor3.7/` | 不解压 |
| `1.修改器/3.7/PC/PvZRHModfied【26-6-28更新】.exe` | `高数羽衫-3.7/修改器/PvZRHModfied【26-6-28更新】.exe` | 不解压 |
| `1.修改器/3.7/其他/其他.APK` | `高数羽衫-3.7/修改器/其他.APK` | 删除  |

**规则**：`.zip` 文件不解压，整体移动到目标目录下；`.APK` 文件直接删除；`.exe` 文件不解压直接移动。

## 二、2.二创植物 → 植物MOD

源路径模式：`2.二创植物/{版本}/*.zip`

**版本 `3.6（兼容3.6.1）` → `3.6.1`**，共 90 个 zip 文件：

| 源路径 | 目标路径 |
|---|---|
| `2.二创植物/3.6（兼容3.6.1）/AstralGodCabbage-究极星神卷心菜.zip` | `高数羽衫-3.6.1/植物MOD/AstralGodCabbage-究极星神卷心菜/` |
| `2.二创植物/3.6（兼容3.6.1）/BarleyNuts-模仿者坚果.zip` | `高数羽衫-3.6.1/植物MOD/BarleyNuts-模仿者坚果/` |
| ... (共 90 个) | ... |

**版本 `3.7` → `3.7`**，共 91 个 zip 文件（比 3.6 多 `NuclearExcitation-终极坍灭粒子炮台.zip` 和 `ShoreboundDuck-上岸鸭.zip`）：

| 源路径 | 目标路径 |
|---|---|
| `2.二创植物/3.7/AstralGodCabbage-究极星神卷心菜.zip` | `高数羽衫-3.7/植物MOD/AstralGodCabbage-究极星神卷心菜/` |
| ... (共 91 个) | ... |

**规则**：每个 zip 解压到 `{zip名不含扩展名}/` 目录下，该目录名即为 MOD 名称。

```
示例：
AstralGodCabbage-究极星神卷心菜.zip
  ├── BepInEx/core/Il2CppInterop.Common.dll
  ├── BepInEx/core/Il2CppInterop.Runtime.dll
  ├── BepInEx/plugins/AstralGodCabbage.dll    ← MOD 文件
  └── ...（其他文件原样保留）
→ 解压到 高数羽衫-3.7/植物MOD/AstralGodCabbage-究极星神卷心菜/
```
