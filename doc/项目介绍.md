# 植物大战僵尸融合版 Mod 服务器

## 功能概述

作为 PvZRH（植物大战僵尸融合版）Mod 服务器，提供：

1. **Mod 文件浏览** — 按作者、版本、类型查询 mod 列表
2. **Mod 文件下载** — 通过路径和文件名直接下载
3. **Mod 文件夹格式化** — 将夸克网盘下载的原始 mod 文件整理为标准目录结构


## API 接口

### 作者列表

```
GET /api/authors
```

返回所有作者名称数组。

### 版本列表

```
GET /api/versions
```

返回所有游戏版本号数组，如 `["3.6.1", "3.7", "通用"]`。

### Mod 列表

```
GET /api/mods?ver={版本}&author={作者}&type={类型}
```

所有参数可选，支持组合过滤。

**返回示例**：

```json
{
  "game_ver": "3.7",
  "author": "高数羽衫",
  "mod_type": "植物MOD",
  "name_cn": "AstralGodCabbage-究极星神卷心菜",
  "file_name": "AstralGodCabbage.dll",
  "url": "https://pvzrh.zhaocheng.cc:8443/api/path?path=高数羽衫/高数羽衫-3.7/植物MOD/AstralGodCabbage-究极星神卷心菜&name=AstralGodCabbage.dll"
}
```

### 下载 Mod

```
GET /api/path?path={目录路径}&name={文件名}
```

以附件形式返回 mod 文件，包含 `Content-Disposition` 头。

### 格式化 Mod 文件夹

```
GET /api/formatpath?author={作者}
```

对指定作者的 mod 文件夹执行格式化，也可通过 CLI 调用：

```bash
pveRH-mod-server -dir D:\Downloads\PVERH-MOD -format=高数羽衫
pveRH-mod-server -dir D:\Downloads\PVERH-MOD -format=高数羽衫 -dry-run  # 预览
```

## CLI 使用

```bash
# 格式化
pveRH-mod-server -dir D:\Downloads\PVERH-MOD -format=高数羽衫

# 预览模式
pveRH-mod-server -dir D:\Downloads\PVERH-MOD -format=林秋鲑鱼 -dry-run

# 启动服务器
pveRH-mod-server -dir D:\Downloads\PVERH-MOD -listen :8443 -url https://pvzrh.zhaocheng.cc:8443
```

## 各作者格式化详情

每位作者的原始文件夹结构差异较大，映射文档见 `doc/` 目录：

| 作者 | 文档 | 特点 |
|---|---|---|
| 高数羽衫 | `高数羽衫.md` | 9 个分类目录，版本规范化，部分 zip 不解压，含删除项 |
| 林秋鲑鱼 | `林秋鲑鱼.md` | 合集 zip 内部选择性提取，GBK→UTF-8 转码 |
| 慕容孤晴 | `慕容孤晴.md` | 纯植物 MOD，结构最简单 |
| 梧萱梦汐 | `梧萱梦汐.md` | 已半格式化，目录移动 + 变体子目录处理 |
