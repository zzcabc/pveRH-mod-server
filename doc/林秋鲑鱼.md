# 林秋鲑鱼 Mod 文件夹格式化映射

## 源结构概览

林秋鲑鱼的 mod 以 **合集 zip** 形式发布，根目录仅 2 个文件：

| 文件名 | 版本 | 大小 |
|---|---|---|
| `鲑鱼MOD整理 3.6.1 (5.31更新）.zip` | 3.6.1 | ~1592 条目 |
| `鲑鱼MOD整理 3.7 (6.19更新）.zip` | 3.7 | ~1562 条目 |

## Zip 内部结构

两个 zip 内部结构相同（3.7 版多一层顶层容器目录 `鲑鱼MOD整理 3.7 (6.19更新）/`）：

```
BepinEX版本/
├── 1.AllMod集合/              ← 全部 mod 合集（跳过，与分类目录重复）
│   └── BepInEx/
│       ├── core/              ← BepInEx 框架文件
│       └── plugins/
│           ├── *.dll          ← 全部 mod dll
│           └── skin/          ← 全部皮肤文件
├── 2.Mod植物/
│   ├── CatShroom-猫娘胆小菇/BepInEx/plugins/{dll}
│   ├── Doro/BepInEx/plugins/{dll}
│   ├── ElectricSuperGatling-超级电能机枪射手/BepInEx/plugins/{dll}
│   ├── ... (共 ~42 个 mod 目录)
│   └── 隐藏植物/
│       ├── MSalmon-外挂鲑鱼/BepInEx/plugins/{dll}
│       ├── PuffFireSniperPea-火焰狙击小喷菇/BepInEx/plugins/{dll}
│       └── PuffSniperPea-狙击小喷菇/BepInEx/plugins/{dll}
├── 3.Mod僵尸/
│   ├── GGGGPenguin-凑企鹅/BepInEx/plugins/{dll}
│   └── MagicZombie-魔术僵尸/BepInEx/plugins/{dll}
├── 4.ModSkin皮肤/
│   ├── All 皮肤集合/BepInEx/plugins/skin/...   ← 合集（跳过）
│   ├── 卷心菜迫击炮-颜值脸/BepInEx/plugins/{skin}
│   ├── 双子向日葵-金毛狮王/BepInEx/plugins/{skin}
│   └── ... (共 ~22 个独立皮肤目录)
├── 5.Mod关卡/
│   ├── CrazySqualour-严肃窝瓜3/BepInEx/plugins/{dll}
│   └── SuperConveyorBelt2-超级模仿传送带·福祸相依/BepInEx/plugins/{dll}
└── 7.Mod插件/
    ├── AutoPlay-节奏大师自动播放/BepInEx/plugins/{dll}
    ├── MakeDoomSniperGreatAgain-飞翔的死神狙击射手/BepInEx/plugins/{dll}
    └── PotaoExplode-土豆爆炸！/BepInEx/plugins/{dll}
```

## 版本号规范化

| 原始文件名 | 规范化版本 |
|---|---|
| `鲑鱼MOD整理 3.6.1 (5.31更新）.zip` | `3.6.1` |
| `鲑鱼MOD整理 3.7 (6.19更新）.zip` | `3.7` |

版本号从文件名中提取 `3.6.1` / `3.7`。

## 目录映射规则

### Zip 内路径 → 目标路径

| Zip 内分类目录 | 目标 MOD 类型 | 说明 |
|---|---|---|
| `2.Mod植物/{MOD名}/BepInEx/` | `植物MOD` | 包含子目录 `隐藏植物/`，同样归入植物MOD |
| `3.Mod僵尸/{MOD名}/BepInEx/` | `僵尸MOD` | |
| `4.ModSkin皮肤/{MOD名}/BepInEx/` | `皮肤MOD` | 排除 `All 皮肤集合`（合集冗余） |
| `5.Mod关卡/{MOD名}/BepInEx/` | `关卡` | |
| `7.Mod插件/{MOD名}/BepInEx/` | `其他` | 插件/工具类 |
| `1.AllMod集合/BepInEx/` | **跳过** | 全量合集，mod 已在分类目录中 |
| `4.ModSkin皮肤/All 皮肤集合/BepInEx/` | **跳过** | 皮肤合集，已在独立目录中 |
| `BepInEx/core/` 等框架文件 | **跳过** | BepInEx 框架由用户自行安装 |

### 处理方式

- 从 zip 中**选择性提取**分类 mod 目录的内容到目标路径
- 去掉中间的 `BepInEx/` 层级，`plugins/` 下的文件直接放入 `{MOD名}/` 目录
- 保留 `plugins/skin/` 子目录结构（皮肤文件）

### 目标格式

```
林秋鲑鱼/林秋鲑鱼-{版本}/{MOD类型}/{MOD名称}/{MOD文件}
```
